# MIMO Signal Transmission Simulation in Python

This repository contains a Python script that models the complete physical layer signal transmission pipeline of a **Multiple-Input Multiple-Output (MIMO)** wireless communication system using **NumPy**.

## 📌 Project Overview

In a MIMO system, multiple data symbols are transmitted simultaneously over multiple antennas, scattering across the environment before arriving at the receiver array. Mathematically, this complex radio interaction is represented by a linear matrix equation:

$$y = Hx + n$$

Where:

* $y$ is the received signal vector.
* $H$ is the complex channel matrix representing path distortions.
* $x$ is the transmitted symbol vector.
* $n$ is the background additive white Gaussian noise (AWGN).

---

## 📖 Step-by-Step Code Explanation

### Step 1: Importing NumPy

The script begins by importing `numpy` to handle the multidimensional array operations and complex matrix mathematics.

```python
import numpy as np

```

### Step 2: Setting Antenna Array Configurations

The parameters define the dimensions of the system. This sets up a MIMO link consisting of $4$ transmit antennas ($n_t = 4$) and $2$ receive antennas ($n_r = 2$).

```python
# Number of transmit antennas
n_t = 4
# Number of receive antennas
n_r = 2

```

### Step 3: Generating the Complex MIMO Channel Matrix

The script constructs the channel matrix $H$ with dimensions $2 \times 4$ (Receive Antennas $\times$ Transmit Antennas).

```python
# Generating a random MIMO channel matrix
H = np.random.randn(n_r, n_t) + 1j*np.random.randn(n_r, n_t)

```

> 💡 **Technical Note:** Wireless waves have both amplitude scaling and a phase shift. To capture this accurately, the code creates a **complex Gaussian matrix** by adding a real random distribution to an imaginary random distribution (`1j * np.random.randn(...)`). This represents a full Rayleigh fading multi-path environment.

### Step 4: Generating the Complex Transmitted Symbol Vector

Next, the script creates a vector $x$ containing $4$ random values—one distinct signal symbol prepared for each of the $4$ transmit antennas. Like the channel, these are generated as complex numbers.

```python
# Generating a random symbol vector
x = np.random.randn(n_t) + 1j*np.random.randn(n_t)

```

### Step 5: Generating the Complex Channel Noise Vector

Thermal background noise is present at the receiver. A $2$-element complex Gaussian noise vector $n$ is generated, matching the total number of physical receive antennas.

```python
# Generating a noise vector
n = np.random.randn(n_r) + 1j*np.random.randn(n_r)

```

### Step 6: Computing the Received Signal Vector

The transmission is simulated by multiplying the channel matrix $H$ by the transmitted symbol vector $x$ using the matrix multiplication operator (`@`), then adding the noise vector $n$. This results in a $2$-element vector $y$, representing the combined data streams reaching the receive antennas.

```python
# Generating a received signal vector
y = H @ x + n

```

### Step 7: Printing the Vectors

Finally, all elements are printed to the console to inspect the numerical results of the complex data transmission.

```python
print("H =", H)
print("x =", x)
print("n =", n)
print("y =", y)

```
